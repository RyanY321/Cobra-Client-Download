Cobra Client Features

  • Cobra Client has these following features:
    • IP Lookup
    • Tron Antimalware
    • Windows User Tools
    • Tools used for Windows systems to make changes to system
    • PC Shutdown Menus
    • Windows Headless State
    • Cobra Client Logs
    • Custom Discord RPC Status
    • Tools For IP
