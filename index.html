What is Cobra Client?

Cobra Client is an easy to use and download utility program for your PC

What can Cobra Client do?

Cobra Client can do many things such as:

  • IP Lookup
  • DNS Clear
  • DNS Tables
  • IP Config
  • Tron Anti-Malware
  • PC Shutdown options
  • Tracert

How do I use Cobra Client?

To ensure the best use of Cobra Client we reccomend putting the file "CobraInstaller.exe" into your Downloads then running the installer

once the installer has been run then you will be a new file in your Downloads called "CobraLauncher.exe"
