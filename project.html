Project Cobra Client

The project Cobra Client was created by two small developers after they had made some tools like this for a smaller organization they made called Melior Media.

Who does Cobra Client belong to?

The copyright of Cobra Client belongs to Melior Media, as this is a Melior Media product.

Do the developers of Cobra Client make any custom programs?

In most cases we do not make any custom programs for anyone. If you would like to make any programs like this then we are free to teach the basics of the code to you. Or you could code another program of your own and work to be a developer of Melior Media, or Cobra Client.
